#  IdeaVault – Startup Idea Sharing Platform

🔗 Live Site: https://your-live-link.vercel.app

## 📌 Project Overview

IdeaVault is a modern startup idea sharing platform where innovators can publish ideas, explore community-driven concepts, and engage through discussions and feedback. The platform encourages collaboration, creativity, and idea validation through user interactions.

---

## ✨ Key Features

*  Secure Authentication (Email/Password & Google Login)
*  Share and manage startup ideas with full CRUD functionality
*  Interactive comment system with edit and delete support
*  Search ideas by title and filter by category
*  Personal dashboard for managing ideas and interactions
*  Fully responsive for Mobile, Tablet, and Desktop devices
